Installation
------------

Symbolic link 'digital_ETRI' to cell-library folder,

    $ ln -s digital_ETRI050_m2d digital_ETRI

QFlow for ETRI050 tech.:

    $ cd ./Tools
    $ chmod +x qflow-1.4.100_etri050_build.sh
    $ ./qflow-1.4.100_etri050_build.sh
